# 📝 TODO LIST 앱 만들기

### 📌 프로젝트 소개
Vue.js를 사용해 제작한 TODO LIST 어플리케이션

<br>

### 📌 기능 구현
✔️ 모달을 사용해 알림창 제작

✔️ localStorage API를 이용해서 데이터 저장

✔️ vuex로 상태관리 적용

✔️ store 모듈화 적용

<br>

### 📌 시연 영상
- 리스트 추가하는 기능

- input이 비어있을 때의 알림 처리 기능

- 리스트 완료 처리하는 기능

- 리스트 삭제하는 기능

- 리스트 전체 삭제하는 기능
